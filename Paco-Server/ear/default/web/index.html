Dear EU & Switzerland Users of Paco

Unfortunately, as of this time, Paco cannot be used in the EU or Switzerland. The reason is that I have yet to determine whether Paco's data collection and retention model conforms to the expected model for the international agreements around data handling. Until a determination has been made about what is needed, if anything, to conform to the new regulations, I cannot sign onto the EU-US Privacy Shield agreement. I am working to resolve this and will update this message as soon as I have it figured out. Please remember that Paco is offered for free by one individual, me, not an organization, and I will do my best to implement a solution if one exists.

However, since Paco is free and open source, it is possible to run your own server in an EU datacenter. Please contact bob@pacoapp.com to find out more about this option.
